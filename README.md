# UDP Server Package

A lightweight and easy-to-use Python package for receiving and inspecting UDP data.  
This package provides a simple `UDPServerNode` class that listens on a specified UDP port and prints all incoming data in hexadecimal format. It is ideal for debugging network protocols, embedded systems communication, IoT devices, and raw datagram payloads.
